# Snake Game

This project includes two Snake game implementations:

- `snake.cpp` — a Windows console version written in C++.
- `index.html`, `styles.css`, `script.js` — a polished browser frontend version.

## Play the web version
1. Open `index.html` in your browser.
2. Use arrow keys or WASD to move.
3. Press Space to pause and Restart to replay.

## Build and run the C++ version
1. Open PowerShell in the project folder.
2. Compile:
   ```powershell
   g++ snake.cpp -o snake.exe
   ```
3. Run:
   ```powershell
   .\snake.exe
   ```

## Notes
- The web frontend does not require any external dependencies.
- The C++ version uses native Windows console functions and is compatible with typical Windows compilers.
